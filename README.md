# MoCapTools
Niema's tools for handling motion and gesture files. Initially, this will likely only contain tools for the [Biovision Hierarchical Data (BVH)](https://en.wikipedia.org/wiki/List_of_motion_and_gesture_file_formats#BVA_and_BVH_file_formats) file format.

# Motion and Gesture File Formats

## Biovision Hierarchical Data (BVH)

Example BVH files can be found in the [Ubisoft La Forge Animation Dataset ("LAFAN1")](https://github.com/ubisoft/ubisoft-laforge-animation-dataset). Helpful resources:

* [https://research.cs.wisc.edu/graphics/Courses/cs-838-1999/Jeff/BVH.html](https://research.cs.wisc.edu/graphics/Courses/cs-838-1999/Jeff/BVH.html)
* [https://www.cs.cityu.edu.hk/~howard/Teaching/CS4185-5185-2007-SemA/Group12/BVH.html](https://www.cs.cityu.edu.hk/~howard/Teaching/CS4185-5185-2007-SemA/Group12/BVH.html)
* [https://docs.rs/bvh_anim/latest/bvh_anim/](https://docs.rs/bvh_anim/latest/bvh_anim/)
